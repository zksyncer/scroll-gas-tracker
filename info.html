Scroll's Bernoulli Upgrade

Scheduled for April 29th, 2024, the Bernoulli upgrade aims to reduce transaction costs by 10x (depending on L1 gas prices). It introduces EIP-4844 data blobs for L1 data availability and the SHA2-256 precompile.

Key Features:

  • EIP-4844 support: Utilizes "blobs" for data transmission, reducing gas costs for data availability.
  • SHA2-256 precompiled contract support: Enables more efficient cryptographic processing.

Technical Details:

The upgrade includes changes to contracts, nodes, and ZKEVM circuits. Contracts will now accept batches with either calldata or blob encoding. Nodes (l2geth) will be updated to v5.3.0, and ZKEVM circuits to v0.10.3.

Timeline:

  • April 15th, 2024: Contract upgrade proposal initiated
  • April 22nd, 2024: New l2geth version release
  • April 29th, 2024: Contract upgrade execution on Scroll Mainnet
